# Telegram AI Bot

Telegram-бот на **aiogram 3** + **OpenAI API** с поддержкой переключаемых режимов (системных промптов), памятью диалога и генерацией видео через **Sora-2**.

## Возможности

- **Чат с GPT** — отправьте текст, бот ответит с учётом контекста последних сообщений
- **Режимы (системные промпты)** — переключение через `/mode`: помощник, разработчик, аналитик и любые свои
- **Генерация изображений** — команда `/image` создаёт картинку через DALL·E и отправляет в чат
- **Генерация видео** — команда `/video` запускает Sora-2 и отправляет готовый MP4 в чат
- **Стоимость запроса** — после каждого ответа показываются токены и цена в рублях (курс ЦБ РФ)
- **Логирование** — консоль + ротируемый файл `logs/bot.log`
- **Proxy API** — для чата и видео можно использовать разные endpoint'ы и ключи

## Структура проекта

## Быстрый старт

### 1. Клонируйте и создайте виртуальное окружение

```bash
git clone <repo-url> && cd Tg_bot_ai1
python -m venv venv
```

Активация:

```bash
# Windows
venv\Scripts\activate

# Linux / macOS
source venv/bin/activate
```

### 2. Установите зависимости

```bash
pip install -r requirements.txt
```

### 3. Создайте файл `.env`

```dotenv
# Telegram
BOT_TOKEN=123456:ABC-...

# OpenAI (чат) — можно через прокси
OPENAI_API_KEY=sk-...
OPENAI_BASE_URL=https://api.openai.com/v1
MODEL_NAME=gpt-5-mini-2025-08-07
MAX_HISTORY=10

# Sora (видео) — отдельные credentials, если прокси не поддерживает /v1/videos
SORA_API_KEY=sk-...
SORA_BASE_URL=https://api.openai.com/v1
SORA_MODEL=sora-2
SORA_SIZE=1280x720
SORA_SECONDS=12
SORA_POLL_INTERVAL_MS=10000

# DALL·E (изображения) — работает через прокси
DALLE_MODEL=dall-e-2
DALLE_SIZE=1024x1024
DALLE_QUALITY=standard
DALLE_PRICE_PER_IMAGE=0.02

# Цены (USD)
PRICE_INPUT_PER_1M=0.15
PRICE_OUTPUT_PER_1M=0.60
SORA_PRICE_PER_SEC=0.10

# Логирование
LOG_LEVEL=INFO
LOG_FILE=bot.log
```

> Если `SORA_API_KEY` / `SORA_BASE_URL` не указаны, используются `OPENAI_API_KEY` / `OPENAI_BASE_URL`.

### 4. Запустите бота

```bash
python main.py
```

## Команды бота

| Команда | Описание |
|---------|----------|
| `/start` | Приветствие, текущий режим, список команд |
| `/mode` | Выбор режима (inline-клавиатура) |
| `/reset` | Очистка истории диалога |
| `/image <описание>` | Генерация изображения через DALL·E |
| `/video <описание>` | Генерация видео через Sora-2 |
| _любой текст_ | Отправка в GPT с контекстом и отображением стоимости |

## Режимы (`prompts.json`)

Файл содержит режимы с системными промптами. Добавить новый — вписать объект в `"prompts"`:

```json
{
  "default_prompt": "assistant",
  "prompts": {
    "assistant": {
      "name": "Обычный помощник",
      "description": "Дружелюбный ассистент для повседневных задач.",
      "system_prompt": "Ты — полезный и дружелюбный помощник."
    },
    "developer": {
      "name": "Опытный разработчик",
      "description": "Помогает писать и объяснять код.",
      "system_prompt": "Ты — опытный разработчик."
    }
  }
}
```

При смене режима через `/mode` история диалога сбрасывается.

## Стоимость запросов

После каждого ответа бот показывает:

```
📊 Токены: 342 вх / 128 вых
💰 Стоимость: $0.000128 ≈ 0.0115 ₽
```

- Курс USD/RUB получается с API ЦБ РФ (`cbr-xml-daily.ru`), кэшируется на 1 час
- Цены за токены и за секунду видео настраиваются в `.env`

## Генерация изображений (DALL·E)

Команда `/image` отправляет промпт в OpenAI Images API:

1. Запрашивается изображение в формате `b64_json`
2. Декодируется и отправляется как фото в Telegram
3. Показывается стоимость

| Параметр | Допустимые значения |
|----------|---------------------|
| `DALLE_MODEL` | `dall-e-2`, `dall-e-3` |
| `DALLE_SIZE` | `256x256`, `512x512`, `1024x1024` (DALL·E 2) / `1024x1024`, `1024x1792`, `1792x1024` (DALL·E 3) |
| `DALLE_QUALITY` | `standard`, `hd` (только DALL·E 3) |

> DALL·E работает через прокси (тот же `OPENAI_BASE_URL`).

## Генерация видео (Sora-2)

Команда `/video` отправляет промпт в OpenAI Videos API:

1. Создаётся задача, SDK опрашивает статус до завершения
2. Скачивается MP4, отправляется в Telegram
3. Временный файл удаляется
4. Показывается стоимость

Доступные параметры:

| Параметр | Допустимые значения |
|----------|---------------------|
| `SORA_MODEL` | `sora-2`, `sora-2-pro` |
| `SORA_SIZE` | `720x1280`, `1280x720`, `1024x1792`, `1792x1024` |
| `SORA_SECONDS` | `4`, `8`, `12` |

> Для Sora нужен прямой ключ OpenAI (Tier 2+). Большинство прокси не поддерживают `/v1/videos`.

## Логирование

- Формат: `2026-03-28 15:30:00 │ INFO     │ __main__ │ сообщение`
- Консоль + файл `logs/bot.log` (ротация 5 МБ, 3 бэкапа)
- Уровень настраивается через `LOG_LEVEL` (`DEBUG` покажет содержимое каждого сообщения)
- Шум от aiogram / openai / httpx подавлен до WARNING
